# Quickstart
Need Python 3.5 interpreter onboard
need json library onboard
Need json datafile on the operational directory 
(for example 
https://op.mos.ru/EHDWSREST/catalog/export/get?id=290344)

```bash

$ python pprint_json.py <path to file>
```
